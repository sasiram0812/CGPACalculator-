# 🎓 CGPA Calculator (React)

A user-friendly **CGPA Calculator** built using React.js that helps students easily calculate their GPA and CGPA semester-wise with accurate results.

## 🌐 Live Demo

👉 https://cgpacalculatorreact.netlify.app/

## 🛠️ Tech Stack

* ⚛️ React.js (Frontend)
* 🎨 CSS (Styling)
* 🌍 Netlify (Deployment)

## ✨ Features

* ➕ Add multiple semesters dynamically
* 📚 Add subjects with credits and grades
* ❌ Remove subjects and semesters
* 📊 Calculate GPA for each semester
* 🎯 Calculate overall CGPA instantly
* 🔄 Real-time updates and results
* 📱 Responsive design

## 📐 Formula Used

* **CPA = Credit × Grade**
* **GPA = Total Credit Points / Total Credits**
* **CGPA = Sum of all Semester GPAs / Number of Semesters**
